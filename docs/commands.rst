Commands
========

Here you can find a description of all ``Nomade`` commands:

Version
-------

Reference:

    $ nomade version

Init
----

Reference:

    $ nomade init

Migrate
-------

Reference:

    $ nomade migrate "My First Migration"

Upgrade
-------

References:

    $ nomade upgrade head

    $ nomade upgrade 1

Downgrade
---------

References:

    $ nomade downgrade tail

    $ nomade downgrade 1

History
-------

Reference:

    $ nomade history

Current
-------

Reference:

    $ nomade current

Help
----

Reference:

    $ nomade --help
